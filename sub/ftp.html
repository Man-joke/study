Visual Studio에서 FTP 접속 방법

패키지 FTP-simple을 Install

F1눌러 도움말 활성화 FTP-simple-config FTP connecting setting 선택

 [
    {
        "name": "NAME",
        "host": "123.123.123.123",
        "port": 포트번호,
        "type": "ftp",
        "username": "username",
        "password": "password",
        "path": "/www",
        "autosave": true, <true면 실서버에 바로적용, false면 파일을 따로 적용>
        "confirm": false <true바꿀때마다 alret창으로 확인받음, false면 확인받지 않고 바로 저장>
    }
  ],

도움말 활성화 해서 FTP-simple remote 실행 하고 해당 FTP 접속
